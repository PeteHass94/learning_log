Implementing a webpage

  • Welcome to books website was illustrated in the Practical 0 file here
  • It was a order form that would be used to obtain user information and display their book orders from the website.

Construction of a form in HTML

Most information was found from www.w3schools.com/html/html_forms.asp

  • The Form used the HTML tag "form"
  • The Text box to fill in used the HTML tag "input" with "type="text""
  • The radio buttons to click in used the HTML tag "input" with "type="radio""
    • Where all radio buttons had the same name, to allow for only 1 to be clicked at a time.
  • The check box to click in used the HTML tag "input" with "type="checkbox""
  • The Text box to fill in the comments used the HTML tag "textarea" with dimensions outlined

Construction of a table in HTML

Most information was found from Powerpoint presentation 3 Hyper Text Markup Language

  • HTML tag table was used with a border thickness of 3 and coloured black to make similar to the practicals illustation.
  • The 4 rows of the table were created using the HTML tag "tr"
  • 4 table headers were created using the HTML tag "th" inside the first row
  • Each row after that added data using the HTML tag "td" this worked in a left to right manor
